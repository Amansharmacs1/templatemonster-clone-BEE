# 🎨 TemplateMonster Landing Page Clone

This repository contains an HTML and CSS clone of a landing page from TemplateMonster, created as part of my **Backend Engineering (BEE)** coursework. The goal of this assignment was to recreate the webpage's layout, styling, and responsive design using only HTML and CSS.

## 🚀 Features

* Responsive landing page layout
* Semantic HTML structure
* Modern CSS styling
* Flexbox and Grid-based layouts
* Clean and organized code

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📁 Project Structure


TemplateMonster-Clone/

│── styles.css
└── assets/
    ├── images
    └── icons


## 🎯 Project Objective

* Practice semantic HTML.
* Improve CSS layout and styling skills.
* Understand responsive web design principles.
* Recreate a real-world landing page from an existing design.
